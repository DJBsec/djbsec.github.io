## Windows Snipping Tool Vulnerability Exposes NTLM Hashes

<span class="badge badge-primary" style="font-size: 0.7rem;">Vulnerability</span>
A vulnerability in the Windows Snipping Tool could expose NTLM hashes when processing certain files. Attackers can exploit this flaw to capture authentication credentials. NTLM hashes can then be used in pass-the-hash attacks to gain access to systems. The issue highlights risks in seemingly low-risk applications. Organizations should apply updates and restrict NTLM usage where possible.
[Read More](https://cybersecuritynews.com/windows-snipping-tool-ntlm-hash/)
